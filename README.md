## Примітки у файлах локалізації

- `#(&)`: потрібне переписання наново;
- `#(&?)`: потрібна ревізія чи вичитка; можливо, треба переписати;
- `#(*)`: текст занадто довгий і не вміщається; скоротити.
- `#((+))`: де перекладач спинився в перекладі
